# DNS

Простое Windows-приложение для преобразования доменного имени в IP-адрес.

## Функции

-   Ввод домена в поле
-   Нажатие кнопки "Найти IP"
-   Отображение IP-адреса (или ошибки) в поле ниже

## Требования

-   Windows
-   Компилятор C++
-   Библиотека Boost

## Сборка

1.  Установить Boost
2.  Скомпилируйте с линковкой: `ws2_32.lib`

        `g++ main.cpp -o dns_resolver.exe -lws2_32 -lboost_system`

    -   флаг -o дает название файлу
    -   -lws2_32 говорит подключить библиотеку ws2_32.lib
    -   -lboost_system подрубает вторую библиотеку
